# Temenos

A simple web page that displays a greeting, the current date and time, and the live Temenos (TEMN.SW) stock price.

## Requirements

- Python 3.x (no external dependencies)

## Usage

```bash
python3 server.py
```

The site will be available at [http://localhost:4867](http://localhost:4867).

## Features

- Hello Temenos World greeting
- Live date and time (updated on each page load)
- Live TEMN.SW stock price fetched from Yahoo Finance
